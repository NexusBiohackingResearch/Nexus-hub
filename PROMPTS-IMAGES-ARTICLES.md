# NEXUS — Prompts d'images pour les articles

Génère chaque image avec ton pipeline (gpt-image-2, logo en référence pour verrouiller la DA).
**Enregistre chaque image sous le nom exact indiqué**, dans le dossier :
`assets/images/articles/`

Le site va automatiquement :
- utiliser `…-cover.png` comme photo de couverture de l'article **et** vignette sur l'accueil,
- insérer `…-1.png`, `…-2.png`, `…-3.png` aux emplacements [IMAGE 1/2/3] dans le texte.

Format conseillé : **couverture 1200×630 (paysage)**, **images internes 1200×800**. Si une image manque, le site masque proprement l'emplacement (aucune image cassée).
